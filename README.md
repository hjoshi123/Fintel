Project Name: FinTel

Decoding Market Moves Through Sentiment Analysis is a sophisticated project designed to harness the vast amounts of unstructured data from financial news sources and social media platforms like Reddit. This project aims to aggregate and analyze the myriad discussions and opinions expressed about stocks across various companies, providing real-time sentiment analysis. The tool categorizes sentiments into bullish, bearish, and neutral. This categorization helps investors gauge the prevailing market sentiment toward particular stocks or the market as a whole.
